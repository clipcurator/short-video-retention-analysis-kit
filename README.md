# Short Video Retention Analysis Kit

English | [简体中文](README.zh-CN.md)

Retention review templates for diagnosing hooks, pacing, drop-off moments, captions, and short video structure.

This repository is a neutral, practical resource for post-publish retention analysis for short videos, hooks, pacing, captions, and clip structure. It is designed for teams that want repeatable decisions, clearer review, and less production guesswork.

## Why This Repository Exists

A short video may look polished but still lose viewers quickly. Retention analysis connects watch data to concrete editing decisions: first seconds, pacing, captions, visual change, story payoff, and ending.

## Who This Is For

- growth teams improving short video performance
- editors learning from watch-time data
- creators comparing multiple clip versions

## Core Workflow

1. Define the input and target output
2. Score or brief the key decision points
3. Use one template to make the work reviewable
4. Run a human QA pass before publishing or production
5. Record what changed so the next cycle improves

Start with the guide that matches your current decision point. Then copy one template into your own workspace, fill it in for one project, and keep the review notes for future iteration.
